# 🚀 Basic Machine Learning Projects

This repository contains end-to-end Machine Learning projects covering the most fundamental and widely used algorithms in Data Science and AI.

The goal of this repository is to understand the complete Machine Learning workflow, from data preprocessing and exploratory data analysis (EDA) to model training, evaluation, and prediction.

---

## 📌 Projects Included

### 1️⃣ Linear Regression - Solar Power Generation Prediction
**Objective:** Predict solar energy output using power generation data.

**Concepts Covered:**
- Data Cleaning
- Feature Selection
- Correlation Analysis
- Linear Regression
- R² Score
- MAE, MSE, RMSE

---

### 2️⃣ Logistic Regression - Steel Plate Fault Detection
**Objective:** Predict whether a steel plate contains a specific defect.

**Concepts Covered:**
- Binary Classification
- Logistic Regression
- Confusion Matrix
- Classification Report
- ROC Curve

---

### 3️⃣ K-Nearest Neighbors (KNN) - Music Genre Prediction
**Objective:** Predict music genres using song and artist attributes.

**Concepts Covered:**
- Distance-Based Learning
- KNN Classification
- Feature Scaling
- Accuracy Evaluation

---

### 4️⃣ Decision Tree - Crop Recommendation System
**Objective:** Recommend suitable crops based on soil nutrients and environmental conditions.

**Concepts Covered:**
- Decision Tree Classification
- Feature Importance
- Tree Visualization
- Multiclass Classification

---

### 5️⃣ Random Forest - Employee Promotion Prediction
**Objective:** Predict employee promotions using HR analytics data.

**Concepts Covered:**
- Ensemble Learning
- Random Forest Classification
- Feature Importance
- Confusion Matrix
- Classification Report
